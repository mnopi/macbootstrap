## Install
```shell script
bash -c "$( curl -sL https://raw.githubusercontent.com/j5pu/macbootstrap/master/install.sh )"
bash -c "$( curl -sL "https://raw.githubusercontent.com/j5pu/macbootstrap/master/install.sh?$(date +%s)" )"
```
